# My eslint configs

## Intallation

Install it with `yarn` or `npm`

```bash
yarn add eslint-config-caonulisses
```

Or 

```bash
npm install eslint-config-caonulisses
```

if you're using npm

## Configuration

Open your `.babelrc` and add this: 

```
{
    "extends": "eslint-config-caonulisses"
}
```
